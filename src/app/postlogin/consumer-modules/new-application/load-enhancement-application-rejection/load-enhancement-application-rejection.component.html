description Consumer Declaration for Cancellation of Load Enhancement Application


मैं यह घोषणा करता/करती हूँ कि मैंने अपने विद्युत कनेक्शन हेतु लोड वृद्धि (Load Enhancement) के लिए आवेदन प्रस्तुत किया था। मैं स्वेच्छा एवं बिना किसी दबाव के अपना लोड वृद्धि आवेदन निरस्त (रद्द) कराने हेतु सहमति प्रदान करता/करती हूँ। मैं यह भी स्वीकार करता/करती हूँ कि आवेदन रद्द किए जाने के उपरांत इस संबंध में भविष्य में मेरी ओर से किसी प्रकार का दावा, आपत्ति अथवा विवाद प्रस्तुत नहीं किया जाएगा।
